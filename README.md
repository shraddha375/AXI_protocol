# AXI Interface

## Types of AXI Interface

- AXI Stream
- AXI Lite
- AXI Full

| Signal    | AXI Stream | AXI Lite |
|-----------|------------|----------|
| `clk`     | 1-bit      | 1-bit    |
| `rstn_h`  | 1-bit      | 1-bit    |
| `rs1n`    | 5-bits     | -        |
| `rs2n`    | 5-bits     | -        |
| `rdn`     | 5-bits     | -        |
| `rdd`     | 32-bits    | 32-bits  |
| `wbe`     | 1-bit      | 1-bit    |
